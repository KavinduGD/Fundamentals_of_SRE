# SRE Introduction

## SRE = Software engineering applied to operations.

<img src="./images/sre-1.png" width="800" />

- **Velocity**: How fast can we deliver new features and improvements to our users?
- **Reliability**: How can we ensure that our services are available and performant for our users?
- So SRE is about finding the right balance between velocity and reliability, and using software engineering principles to achieve that balance.
